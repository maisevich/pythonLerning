## Задача
**Создать базу данных работников
отделения больницы.**

Фамилия, имя, должность, отделение, личный номер.

***Модули проекта:***

**User_interface**  - предоставляет пользователю возможность ввести данные для дальнейшого сохранения.
**CSV_creating** - создает файл .csv и записывает в него шапку таблицы.
**File_writing** - содержит методы для записи файла csv и .txt.
